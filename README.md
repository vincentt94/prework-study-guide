# <Prework Study Guide Webpage>

## Description  

This study guide helps prepare bootcamp students with information about HTML, CSS, Git, &Javascript.  Through the use of practical application, students will learn the basics of HTML, CSS, Git, & Javascript to deploy their own version of a study guide to help them retain and learn some fundementals of coding.

## Usage

To utilize this Prework Study Guide, view the notes in each section in regards to HTML, CSS, Git, & Javascript.  Through the use of Chrome Devltools, students can be suggested which topics to study first by checking out the console panel.  Right click the page and go down to inspect.  On the right there is a console panel where a list of topics is displayed and a suggested topic is announced to study.
